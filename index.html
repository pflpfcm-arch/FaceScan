/** * PFL Attendance - Backend API * ปรับปรุงระบบบันทึกแบบแยกส่วนและรองรับการค้นหาข้อมูลวันเดียวกัน (Upsert) */ const SPREADSHEET_ID = "1qxS1TEInY7eeOSuEs11FELHcR17KpeeIHGaqg9ShTjA"; const SS = SpreadsheetApp.openById(SPREADSHEET_ID); const SHEETS = { USERS: "Users", ATTENDANCE: "Attendance", ENTRY_POINTS: "EntryPoints", WORK_POINTS: "WorkPoints", LOGS: "Logs" }; function doGet(e) { const action = e.parameter.action; try { switch (action) { case "getUsers": return contentResponse(getDataFromSheet(SHEETS.USERS)); case "getAttendance": return contentResponse(getDataFromSheet(SHEETS.ATTENDANCE).reverse()); case "getEntryPoints": return contentResponse(getDataFromSheet(SHEETS.ENTRY_POINTS)); case "getWorkPoints": return contentResponse(getDataFromSheet(SHEETS.WORK_POINTS)); default: return contentResponse({ error: "Invalid action" }); } } catch (err) { return contentResponse({ error: err.toString() }); } } function doPost(e) { const action = e.parameter.action; const payload = JSON.parse(e.postData.contents); try { switch (action) { case "saveAttendance": return contentResponse(upsertAttendance(payload)); case "saveUser": return contentResponse(upsertUser(payload)); case "deleteUser": return contentResponse(deleteById(SHEETS.USERS, payload.id, "id")); case "saveEntryPoint": return contentResponse(upsertPoint(SHEETS.ENTRY_POINTS, payload)); case "deleteEntryPoint": return contentResponse(deleteById(SHEETS.ENTRY_POINTS, payload.name, "name")); case "saveWorkPoint": return contentResponse(upsertPoint(SHEETS.WORK_POINTS, payload)); case "deleteWorkPoint": return contentResponse(deleteById(SHEETS.WORK_POINTS, payload.name, "name")); case "saveLog": if (payload.type === 'register') updateUserRegistration(payload.userId, payload.pin, payload.faceDescriptor); return contentResponse(saveToSheet(SHEETS.LOGS, [payload.userId, payload.type, new Date().toISOString(), JSON.stringify(payload)])); default: return contentResponse({ error: "Invalid POST action" }); } } catch (err) { return contentResponse({ error: err.toString() }); } } // --- ฟังก์ชันบันทึกข้อมูลแบบอัปเดตบรรทัดเดิมของวันนั้นๆ --- function upsertAttendance(payload) { const sheet = SS.getSheetByName(SHEETS.ATTENDANCE); const data = sheet.getDataRange().getValues(); // วันที่ปัจจุบันสำหรับค้นหา (dd/MM/yyyy) const now = new Date(); const dateToday = Utilities.formatDate(now, "GMT+7", "dd/MM/yyyy"); const fullTs = Utilities.formatDate(now, "GMT+7", "dd/MM/yyyy, HH:mm"); let rowIndex = -1; // ค้นหา UserId และ วันที่ปัจจุบันในคอลัมน์ J (Index 9) for (let i = 1; i < data.length; i++) { const sheetTimestamp = data[i][9] ? data[i][9].toString() : ""; if (data[i][0] == payload.userId && sheetTimestamp.indexOf(dateToday) === 0) { rowIndex = i + 1; break; } } if (rowIndex > -1) { // กรณีพบข้อมูลของวันนี้แล้ว ให้ทำการ "อัปเดต" เฉพาะส่วนที่ส่งมา if (payload.subType === 'company') { sheet.getRange(rowIndex, 3).setValue(payload.pointCompany); // Col C sheet.getRange(rowIndex, 4).setValue(payload.timeCompany); // Col D } else if (payload.subType === 'work') { sheet.getRange(rowIndex, 5).setValue(payload.pointWork); // Col E sheet.getRange(rowIndex, 6).setValue(payload.timeWork); // Col F } // อัปเดตสถานะล่าสุด และโน้ต if (payload.status !== 'ยังไม่ลงบันทึก') sheet.getRange(rowIndex, 3).setValue(payload.status); sheet.getRange(rowIndex, 8).setValue(payload.note); sheet.getRange(rowIndex, 9).setValue(payload.lat); sheet.getRange(rowIndex, 10).setValue(payload.lng); // ไม่เขียนทับเวลาเริ่มวัน (Keep original date mark, but update full timestamp if needed) sheet.getRange(rowIndex, 11).setValue(fullTs); } else { // กรณีไม่พบข้อมูลของวันนี้ ให้ "เพิ่มแถวใหม่" sheet.appendRow([ payload.userId, payload.nickname, payload.status, payload.pointCompany, payload.timeCompany, payload.pointWork, payload.timeWork, payload.note, payload.lat, payload.lng, fullTs, payload.recordedByAdmin ]); } return { status: "success" }; } function getDataFromSheet(sheetName) { const sheet = SS.getSheetByName(sheetName); if (!sheet) return []; const data = sheet.getDataRange().getValues(); if (data.length <= 1) return []; const headers = data.shift(); return data.map(row => { let obj = {}; headers.forEach((h, i) => obj[h.toString().trim()] = row[i]); return obj; }); } function saveToSheet(sheetName, rowData) { const sheet = SS.getSheetByName(sheetName); sheet.appendRow(rowData); return { status: "success" }; } function upsertUser(payload) { const sheet = SS.getSheetByName(SHEETS.USERS); const data = sheet.getDataRange().getValues(); const headers = data[0]; const idCol = headers.indexOf("id"); let rowIndex = -1; for (let i = 1; i < data.length; i++) { if (data[i][idCol] == payload.id) { rowIndex = i + 1; break; } } if (rowIndex > -1) { sheet.getRange(rowIndex, headers.indexOf("nickname") + 1).setValue(payload.nickname); sheet.getRange(rowIndex, headers.indexOf("role") + 1).setValue(payload.role); } else { const newRow = new Array(headers.length).fill(""); newRow[idCol] = payload.id; newRow[headers.indexOf("nickname")] = payload.nickname; newRow[headers.indexOf("role")] = payload.role; sheet.appendRow(newRow); } return { status: "success" }; } function upsertPoint(sheetName, payload) { const sheet = SS.getSheetByName(sheetName); const data = sheet.getDataRange().getValues(); const nameCol = data[0].indexOf("name"); let rowIndex = -1; for (let i = 1; i < data.length; i++) { if (data[i][nameCol] == payload.name) { rowIndex = i + 1; break; } } const rowData = [payload.name, payload.lat, payload.lng, payload.range]; if (rowIndex > -1) sheet.getRange(rowIndex, 1, 1, 4).setValues([rowData]); else sheet.appendRow(rowData); return { status: "success" }; } function deleteById(sheetName, idValue, idColumnName) { const sheet = SS.getSheetByName(sheetName); const data = sheet.getDataRange().getValues(); const idCol = data[0].indexOf(idColumnName); for (let i = 1; i < data.length; i++) { if (data[i][idCol] == idValue) { sheet.deleteRow(i + 1); return { status: "success" }; } } return { status: "not_found" }; } function updateUserRegistration(userId, hashedPin, faceDescriptor) { const sheet = SS.getSheetByName(SHEETS.USERS); const data = sheet.getDataRange().getValues(); const headers = data[0]; const idCol = headers.indexOf("id"); for (let i = 1; i < data.length; i++) { if (data[i][idCol] == userId) { sheet.getRange(i + 1, headers.indexOf("pin") + 1).setValue(hashedPin); sheet.getRange(i + 1, headers.indexOf("faceDescriptor") + 1).setValue(faceDescriptor); return true; } } return false; } function contentResponse(obj) { return ContentService.createTextOutput(JSON.stringify(obj)).setMimeType(ContentService.MimeType.JSON); }
